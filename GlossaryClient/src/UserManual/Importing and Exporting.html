To import a glossary from a file

If you want to import a new glossary from your computer you can click on File at the menu bar and then click on import,or you can press directly CTRL+I at the same time. Now you have to select a file you've exported before, subsequently press "Open" to confirm the operation.

To export a glossary from a file

If you want to export the glossary you've just written to your computer you can click the File at the menu bar and then click on export,or you can press CTRL+E at the same time. Now you have to select the folder's path and finally save the file.
