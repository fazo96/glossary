To delete a term from the glossary

You have firs to select the term you want to delete then press the "Delete" button and confirm with the "Ok" button if you are sure.

To save the term's meaning

If you want to change the term's meaning already existing or a term's meaning added from you, you have to select the term and modify the meaning using the right text area. When you finished modifying press the "Save" button.

To search a term

You have to write in the search bar(situated at the bottom-left) the term you want to find. While you're writing the list updates with you filter.

To Import a glossary from a file

If you want to import a new glossary from your computer you can click the file menu and then click on import,you can also press CTRL+I. Now you have to select a file you've exported before.
