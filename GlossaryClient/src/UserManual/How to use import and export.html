To Import a glossary from a file

If you want to import a new glossary from your computer you can click the file menu and then click on import,you can also press CTRL+I. Now you have to select a file you've exported before, subsequently press "Open" to confirm the operation.

To Import a glossary from a file

If you want to export the glossary you've just written to your computer you can click the file menu and then click on export,you can also press CTRL+S. Now you have to select the folder where you want to save the new file.
